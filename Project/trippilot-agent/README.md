## Quick Start

```bash
# 1. Start infrastructure (MongoDB, Phoenix, Grafana)
cd observability
docker compose up -d

# 2. Start backend
cd ../backend
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env  # Add your OPENAI_API_KEY
python main.py

# 3. Start frontend
cd ../frontend
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
streamlit run Home.py
```
